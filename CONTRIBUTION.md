# Contributing to Lift A Child Uganda Website

Thank you for considering contributing to this project! This document outlines the process for contributing to the Lift A Child Uganda website.

## Code of Conduct

Please be respectful and considerate of others when contributing to this project.

## How Can I Contribute?

### Reporting Bugs
- Use the GitHub Issues tracker
- Describe the bug in detail
- Include steps to reproduce
- Specify your browser and OS version

### Suggesting Enhancements
- Open an issue with the "enhancement" label
- Clearly describe the feature
- Explain why it would be useful

### Code Contributions
1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Test thoroughly
5. Submit a pull request

## Development Setup

1. Clone the repository
2. Open `index.html` in a browser
3. No build process required (static website)

## Coding Standards

### HTML
- Use semantic HTML5 elements
- Add appropriate ARIA labels
- Keep indentation consistent (2 spaces)

### CSS
- Use CSS Custom Properties for theming
- Follow BEM naming convention for complex components
- Mobile-first responsive design

### JavaScript
- Use vanilla ES6+ JavaScript
- Add comments for complex logic
- Handle errors gracefully

## Pull Request Process

1. Update the README.md if needed
2. Update the CHANGELOG.md with changes
3. Ensure all code follows the standards
4. Request review from maintainers

## Questions?

Feel free to open an issue for any questions about contributing.
