# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 22/01/2026

### Added
- Initial project setup with complete website structure
- Single-page application navigation system
- Responsive design for all device sizes
- Interactive contact and volunteer forms
- Program showcase with hover animations
- Testimonial section with client stories
- Donation integration capabilities
- Mobile-first responsive navigation
- CSS animations and transitions
- External CSS and JavaScript file separation

### Technical Features
- Semantic HTML5 structure
- CSS Grid and Flexbox layouts
- CSS Custom Properties (variables)
- Vanilla JavaScript for interactivity
- Intersection Observer for scroll animations
- Mobile-responsive hamburger menu
- Form validation and submission handling
- GitHub Pages deployment configuration
- Comprehensive project documentation
